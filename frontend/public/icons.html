Creating App Icons for Guest Manager

The following image files need to be created in the public folder:

  • logo192.png - 192x192 pixel PNG image
  • logo512.png - 512x512 pixel PNG image

Note: These files are required for the PWA manifest to work correctly. The manifest.json has been updated to reference these local files instead of external URLs.

Quick Solution

You can use a simple temporary icon or your app logo resized to these dimensions.

Using Image Editing Software

1. Open your logo in an image editor (Photoshop, GIMP, etc.)
2. Resize to 192x192 pixels and save as logo192.png
3. Resize to 512x512 pixels and save as logo512.png
4. Place both files in the public folder

Using Online Tools

You can use online tools like favicon.io to generate all required app icons.
